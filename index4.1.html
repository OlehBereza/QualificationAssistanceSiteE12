KOMENDY CMD





KOMENDY Z KONTAMI W CMD


net user NAME /add utw. konto gdzie 'NAME' jest imię konta
net user NAME informacje o koncie
net user NAME pass utw. konto gdzie 'pass' jest twoje hasło
net user NAME /times:Pn-Wt,15-19 czas kiedy można zalogować się na to konto
net user NAME /comment:”Komentarz” dodatkowo możemy dodać komentarz do wybranego konta
net localgroup (grupa) NAME /add dodać konto do grupy lokalnej
net user NAME /active:yes zrobić aktywne konto
net user NAME /del usunuć konto
